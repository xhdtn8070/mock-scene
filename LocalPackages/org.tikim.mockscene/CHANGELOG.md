# Changelog

All notable changes to `org.tikim.mockscene` will be documented in this file.

## [1.0.0] - 2026-04-16

### Added

- Scene-scoped mock preset model with `MockScenePreset`.
- Scene-owned active preset tracking with `MockSceneSceneSettings`.
- Editor-only play mode resolver with `MockSceneInjector`.
- `Tools > MockScene Manager` window for creating, activating, and copying presets.
- EditMode tests for preset normalization and scene settings synchronization.
- Public package metadata for documentation, changelog, and license discovery.
